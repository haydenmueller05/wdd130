Why I made this website

I want this website to be somewhere where anyone who is lost or stuck in their coding career can see how I got to where I am. I remember when i first started and I felt so lost and I felt like coding was so confusing and so broad. I didnt know where to start, So I want this website to help those know where to start and how to start their own coding career. This website will have my projects, experiences, blog, and things that ive gone through to get here. I know the hardest part about getting into a coding career is starting, I want to help others know how to start.
